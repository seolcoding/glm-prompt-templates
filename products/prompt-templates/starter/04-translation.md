# 번역 템플릿

## 전문 번역

```
당신은 전문 번역가입니다.

다음 텍스트를 [원어]에서 [목표어]로 번역해주세요:

**원문**:
"""
[텍스트]
"""

**번역 가이드**:
- 분야: [기술/비즈니스/법률/의료/마케팅]
- 톤: [격식/비격식/학술]
- 타겟 독자: [누구인지]
- 용어 처리: [원어 유지/한글화/혼합]

**요구사항**:
1. 의역 우선 (자연스러운 흐름)
2. 문화적 맥락 고려
3. 전문 용어 일관성
4. 원문의 뉘앙스 보존

**출력**:
## 번역
[번역문]

## 용어 표 (필요시)
| 원어 | 번역 |
|------|------|

## 번역 노트
[어려웠던 부분이나 대안]
```

## 현지화 (Localization)

```
당신은 현지화 전문가입니다.

다음 콘텐츠를 [국가/지역] 시장에 맞게 현지화해주세요:

**원본 콘텐츠**:
"""
[콘텐츠]
"""

**현지화 요소**:
1. 언어적 현지화
   - 자연스러운 표현
   - 관용구/속어 대체
   - 존댓말/반말 적절히

2. 문화적 현지화
   - 문화 참조 대체
   - 날짜/시간 형식
   - 통화/단위 변환
   - 이미지/색상 의미

3. 법적/규제 적응
   - 현지 법규 준수
   - 필수 표기사항

**출력**:
## 현지화된 콘텐츠
[결과]

## 변경 사항 설명
- [무엇을 왜 변경했는지]
```

## 기술 문서 번역

```
당신은 기술 문서 번역 전문가입니다.

다음 기술 문서를 번역해주세요:

**원문**:
"""
[기술 문서]
"""

**번역 원칙**:
1. 코드/명령어는 원문 유지
2. 기술 용어는 업계 표준 사용
3. UI 요소는 [원문/번역] 병기
4. 링크/URL 유지

**용어 사전** (필요시 추가):
| 영어 | 한국어 |
|------|--------|

**출력 형식**:
번역문 + 용어 주석
```
