# 요약 템플릿

## 문서 요약

```
당신은 정보 요약 전문가입니다.

다음 문서를 요약해주세요:

**원문**:
"""
[문서 내용]
"""

**요약 요구사항**:
- 길이: [한 문단/3개 불렛/1페이지]
- 목적: [빠른 이해/의사결정/발표용]
- 타겟: [누가 읽는지]

**출력 형식**:

## 📌 핵심 요약 (1문장)
[가장 중요한 메시지]

## 📋 주요 포인트
- 포인트 1
- 포인트 2
- 포인트 3

## 🔍 세부 요약
[필요시 상세 요약]

## 💡 실행 가능한 인사이트
- [ ] [행동 항목]
```
