🧵 마케터가 반드시 알아야 할 10가지 ChatGPT 프롬프트

AI를 활용하면 마케팅 콘텐츠 작성 시간을 70% 줄일 수 있습니다.

하지만 "그냥 써봐"라고만 하면 평범한 결과만 나와요.

핵심은 **정확한 프롬프트**입니다.

제가 직접 테스트한 10가지 필수 프롬프트를 공유합니다:

👇

1️⃣ **블로그 포스트 작성**

당신은 10년 경력의 콘텐츠 마케터입니다.

[주제]에 대한 1,500자 블로그 포스트를 작성해주세요.

**타겟 독자**: [구체적 타겟]
**톤**: [전문적/친근함]

→ 복사해서 [대괄호]만 수정하면 됩니다

2️⃣ **이메일 캠페인 (3단계)**

[제품]을 홍보하는 3개 이메일 시퀀스:
1. 인지 - 문제 제기
2. 고려 - 해결책 제시
3. 행동 - 긴급성 + 할인

→ 각 이메일 200자 이내로 작성

3️⃣ **인스타그램 캡션 (3가지 버전)**

[제품]에 대한 캡션:
• 스토리텔링 버전
• 교육적 버전
• 직접 CTA 버전

→ 타겟에 맞는 버전 선택

4️⃣ **SEO 메타 데이터**

[페이지]의 SEO 최적화:
• Title (60자 이내)
• Meta Description (160자 이내)
• H1 태그

→ 클릭률 높은 제목 생성

5️⃣ **A/B 테스트 광고**

[제품]에 대한 2가지 광고 카피:
• 버전 A: 감성 어필
• 버전 B: 데이터 중심

→ 성과 측정 후 승자 선택

---

💡 **프롬프트 작성 팁**:

• 구체적으로: "좋은 글" ❌ → "1,500자 블로그 포스트" ✅
• 맥락 제공: 타겟 독자, 톤, 목표 명시
• 예시 추가: "이런 스타일로: [예시]"
