# 비즈니스 모델

## 1. 디지털 제품 (Digital Products)

### 템플릿
- **가격대**: $9-99 일회성
- **마진율**: 95%+
- **확장성**: 매우 높음
- **예시**: Notion 템플릿, Excel 시트, Figma 키트

### 이북/가이드
- **가격대**: $19-199 일회성
- **마진율**: 90%+
- **확장성**: 높음
- **예시**: PDF 가이드, 단계별 튜토리얼

### 코스
- **가격대**: $49-999 일회성
- **마진율**: 85%+
- **확장성**: 중간
- **예시**: 비디오 코스, 워크샵

## 2. 구독 서비스 (Subscriptions)

### 뉴스레터
- **가격대**: $5-29/월
- **마진율**: 90%+
- **확장성**: 높음
- **예시**: 프리미엀 뉴스레터, 독점 콘텐츠

### 마이크로 SaaS
- **가격대**: $9-99/월
- **마진율**: 80%+
- **확장성**: 높음
- **예시**: 자동화 도구, 분석 대시보드

## 3. 제품화된 서비스 (Productized Services)

### 설정 서비스
- **가격대**: $99-999 일회성
- **마진율**: 60%+
- **확장성**: 낮음
- **예시**: 템플릿 커스터마이징, 설정 도움

### 월간 리텐션
- **가격대**: $99-499/월
- **마진율**: 50%+
- **확장성**: 낮음
- **예시**: 지속적인 관리, 업데이트

## 수익화 사다리 (Monetization Ladder)

```
무료 (Free) → 저가 ($9-29) → 핵심 ($49-199) → 업셀 ($199-999) → 구독 ($9-99/월)
```

### 예시: AI 프롬프트 템플릿
1. **무료**: 프롬프트 작성 체크리스트 PDF
2. **저가**: "50개 필수 프롬프트" ($19)
3. **핵심**: "프롬프트 엔지니어링 마스터 클래스" ($97)
4. **업셀**: "1:1 프롬프트 컨설팅 세션" ($297)
5. **구독**: "월간 신규 프롬프트 + 업데이트" ($19/월)

- 자동 갱신 공개

### 서비스
- 예상 결과물 명시
- 타임라인 현실적 설정
