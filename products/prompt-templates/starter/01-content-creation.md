# 콘텐츠 제작 템플릿

## 블로그 글 작성

```
당신은 [분야] 전문 콘텐츠 크리에이터입니다.

다음 정보를 바탕으로 블로그 글을 작성해주세요:

**주제**: [주제]
**타겟 독자**: [누구인지]
**목표**: [독자가 무엇을 얻어야 하는지]
**톤**: [전문적/친근함/유머러스]
**길이**: [글자 수]

**구조**:
1. 흥미로운 시작 (문제 제기)
2. 본론 (3-5개 섹션)
3. 실천 가능한 결론
4. CTA (다음 단계)

**요구사항**:
- SEO를 고려한 제목
- 소제목 활용
- 읽기 쉬운 문단 구성
- 예시나 사례 포함
```

## 인스타그램 캡션

```
당신은 소셜 미디어 전문가입니다.

[제품/서비스/주제]에 대한 인스타그램 캡션을 작성해주세요:

**브랜드 톤**: [캐주얼/전문적/유머러스]
**목표**: [참여/판매/브랜드 인지]
**해시태그 수**: [개수]

**포함할 요소**:
- 훅 (첫 줄에서 관심 끌기)
- 가치 제공
- CTA
- 관련 해시태그

3가지 버전을 만들어주세요:
1. 스토리텔링 버전
2. 교육적 버전
3. 감성적 버전
```

## 유튜브 스크립트

```
당신은 유튜브 크리에이터입니다.

**영상 주제**: [주제]
**영상 길이**: [분]
**타겟 시청자**: [누구인지]
**썸네일 클릭을 유도할 훅**: [무엇인지]

다음 구조로 스크립트를 작성해주세요:

1. **오프닝 (30초)**
   - 훅으로 시작
   - 이 영상에서 얻을 것 미리보기
   - 구독/좋아요 요청

2. **본론**
   - 3-5개 핵심 포인트
   - 각 포인트당 예시/데모
   - 시청자 질문 예상하여 답변

3. **결론 (30초)**
   - 핵심 요약
   - 다음 영상 예고
   - CTA

**스타일**: [대화형/강의형/vlog형]
```
