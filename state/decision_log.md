### 결정사항
1. **TypeScript 선택**: 간단한 구조와 타입 안전성을 위해 TypeScript 채택
2. **JSON 파일 저장**: 초기 단계에서 복잡성을 줄이기 위해 JSON 파일 사용
3. **로컬 우선**: 외부 의존성을 최소화하여 로컬에서 실행 가능한 구조
