# LinkedIn 가이드

## 🚀 빠른 설정 (5분)

### 1️⃣ 계정 생성
1. https://www.linkedin.com/ 접속
2. "가입" 클릭
3. 이름, 이메일, 비밀번호 입력
4. 직장:
   - **회사**: "AI Productivity Lab" 또는 "프리랜서"
   - **직책**: "AI Consultant" 또는 "Content Creator"
   - **지역**: "서울, 대한민국"

### 2️⃣ 프로필 최적화
1. **프로필 사진**: 전문적인 이미지 (LinkedIn 권장)
2. **배경 이미지**: AI/생산성 관련
3. **헤드라인**:
   ```
   AI 프롬프트 엔지니어링 전문가 | 생산성 향상 70% | ChatGPT/Claude 최적화
   ```
4. **요약**:
   ```
   🚀 10년 경력 소프트웨어 개발자 & AI 컨설턴트
   📝 ChatGPT/Claude 프롬프트 템플릿 제작
   🎯 마케터·개발자·콘텐츠 크리에이터 생산성 향상
   💡 무료 프롬프트 가이드: [링크]
   ```

### 3️⃣ 첫 번째 포스트
```bash
./scripts/auto-post-browser.sh linkedin
```
