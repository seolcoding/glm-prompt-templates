# 30분 만에 ChatGPT로 이메일 캠페인 완성하기

> **실전 가이드**: 바로 복사해서 사용할 수 있는 템플릿 포함

## 🎯 왜 이메일 캠페인이 중요한가?

이메일 마케팅의 ROI는 **$42/$1**입니다. 즉, 1달러 투자하면 42달러 수익.하지만 매일 이메일을 쓰는 데만 몇 시간을 쓰시나요?

ChatGPT를 활용하면 **30분** 만에 일주일 치 이메일 캠페인을 완성할 수 있습니다.

---

## 📝 3단계 이메일 캠페인 템플릿

### 1단계: 인지 (Awareness)

```
당신은 10년 경력의 이메일 마케터입니다.

[제품/서비스]에 대한 첫 번째 이메일을 작성해주세요.

**목적**: 문제 인지,**타겟**: [구체적 타겟]
**톤**: 공감적

**구조**:
1. 후킹 제목 (클릭 유도)
2. 문제 제기
3. 해결책 힌트
4. 다음 이메일 예고

**길이**: 150자 이내
```

### 2단계: 고려 (Consideration)

```
[제품/서비스]의 핵심 기능을 소개하는 이메일.

**목적**: 솔루션 제시
**포함할 내용**:
- 핵심 기능 3개
- 사용자 후기 1개
- 경쟁사 대비 장점

**길이**: 200자 이내
```

### 3단계: 행동 (Action)

```
[제품/서비스] 구매/가입을 유도하는 이메일.

**목적**: 전환
**포함할 내용**:
- 한정 할인 (24시간)
- 긴급성 부여
- 명확한 CTA

**길이**: 150자 이내
```
