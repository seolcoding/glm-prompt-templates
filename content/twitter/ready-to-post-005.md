🧵 블로거를 위한 ChatGPT 워크플로우 (복사해서 쓰세요)

블로그 쓰는 데 하루 종일 걸리시나요?

이 워크플로우로 1시간 만에 7개 포스트를 씁니다.

👇

## 📝 1단계: 키워드 리서치

```
[주제]에 대한 블로그 키워드 10개 추천해줘세요.

기준:
- 월 검색량 1,000+
- 경쟁도 낮음
- 클릭당 지불 가능

각 키워드별로:
1. 월 검색량
2. 추천 제목
3. 콘텐츠 각도
```

→ 1분 소요

## 📝 2단계: 아웃라인 생성
```
[키워드]에 대한 블로그 포스트 아웃라인을 작성해주세요.

구조:
1. 후킹 제목 (3가지 옵션)
2. 소개 (후킹 문장 + 문제 제기)
3. 본론 (3-5개 섹션)
4. 결론 + CTA

각 섹션별 핵심 포인트 포함
```

→ 3분 소요

## 📝 3단계: 본문 작성
```
아웃라인을 바탕으로 1,500자 블로그 포스트를 작성해주세요.

요구사항:
- SEO 키워드 자연스럽게 포함
- 짧은 문단 (3-4문장)
- 소제목 사용
- 실천 가능한 팁 포함
- 한국어 존댓말
```

→ 10분 소요

## 📝 4단계: SEO 메타데이터
```
이 블로그 포스트의 SEO 메타데이터를 작성해주세요:
1. Title (60자 이내, 키워드 포함)
2. Meta Description (160자 이내)
3. Slug (URL)
4. 태그 10개
```

→ 2분 소요

---

## ⚡ 전체 시간: 16분

기존 방식: 3-4시간 → **시간 절약: 90%+**

💡 **프로 팁**:

• 2-3단계 반복으로 일괄 생성
• 4단계에서 포맷팅 자동화
