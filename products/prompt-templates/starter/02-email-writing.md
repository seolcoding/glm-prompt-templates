# 이메일 작성 템플릿

## 비즈니스 이메일

```
당신은 전문 비즈니스 커뮤니케이터입니다.

다음 정보로 이메일을 작성해주세요:

**받는 사람**: [이름/직함]
**관계**: [처음/기존/상사/클라이언트]
**목적**: [요청/제안/보고/사과/감사]
**핵심 내용**: [무엇에 관한 것인지]
**원하는 결과**: [답변/승인/미팅/등]

**형식**:
- 제목: [명확하고 구체적으로]
- 인사: [관계에 맞게]
- 본문: [간결하게]
- CTA: [다음 단계]
- 맺음말

**제약사항**:
- 길이: [3문장 이내/상세하게]
- 톤: [정중함/직접적/부드러움]
```

## 영업 이메일 (콜드 메일)

```
당신은 B2B 영업 전문가입니다.

**타겟**: [직함/산업/기업 규모]
**제품/서비스**: [무엇을 파는지]
**핵심 가치**: [고객이 얻을 이익]
**차별점**: [경쟁사 대비]

콜드 메일을 작성해주세요:

**구조**:
1. 개인화된 오프닝 (관련성 입증)
2. 문제 제기 (고객의 pain point)
3. 솔루션 제시 (우리 제품)
4. 사회적 증거 (고객 사례/수치)
5. 낮은 장벽 CTA (15분 통화 등)

**제약사항**:
- 제목: 50자 이내, 클릭 유도
- 본문: 150단어 이내
- 무엇을 요청하는지 명확히
```

## 후속 이메일

```
당신은 영업 후속관리 전문가입니다.

**상황**: [미팅 후/제안 후/연락 없음/등]
**경과 시간**: [며칠/몇 주]
**이전 대화 내용**: [핵심 요약]
**다음 단계**: [무엇을 원하는지]

후속 이메일을 작성해주세요:

**요구사항**:
- 가치 추가 (새로운 정보/자료)
- 구체적인 다음 단계 제안
- 압박감 없는 톤
- 답변하기 쉬운 질문 포함

3가지 버전:
1. 부드러운 리마인더
2. 가치 추가 버전
3. 긴급성 부여 버전
```
