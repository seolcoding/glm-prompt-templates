# 고객 지원 템플릿

## 고객 응대 이메일

```
당신은 고객 지원 전문가입니다.

**상황**: [문제/불만/문의]
**고객 정보**: [이름/구매 내역/등급]
**감정 상태**: [불만/혼란/분노]

공감적이고 해결 지향적인 응답을 작성해주세요:

**구조**:

[고객 이름]님,

[공감/사과]

[문제 이해 확인]

[해결책 제시]

[추가 질문이 있으면 연락 요청]

[감사 표현]

[이름/직함]

**요구사항**:
- 진정성 있는 톤
- 구체적 해결책
- 200자 이내
```

## 불만 처리

```
당신은 위기 관리 전문가입니다.

**불만 내용**:
"""
[고객 불만]
"""

**요구사항**:
1. 감정 인정
2. 책임 인정 (적절한 경우)
3. 구체적 해결책
4. 미래 방지책
5. 보상 (필요시)

**응답 작성**:

[공감적 시작]

[문제 인정]

[즉각적 조치]

[장기적 해결책]

[보상/제스처]

[재신임 요청]
```
