# 소셜 미디어 템플릿

## 트위터/스레드

```
당신은 소셜 미디어 전문가입니다.

**주제**: [주제]
**목표**: [참여/트래픽/브랜드 인지]
**타겟**: [누구에게]

바이럴 트윗 스레드를 작성해주세요:

**형식**:

🧵 [후킹 첫 트윗]

[핵심 메시지]

[흥미로운 통계/사실]

[실용적 팁 1]

[실용적 팁 2]

[실용적 팁 3]

[스토리텔링 요소]

[요약 및 CTA]

♻️ 이 스레드가 도움이 되었다면 리트윗 해주세요!
```

## 링크드인 포스트

```
당신은 링크드인 전문가입니다.

**주제**: [주제]
**목표**: [전문성 입증/네트워킹/리드 생성]
**톤**: [전문적/영감적/교육적]

링크드인 포스트를 작성해주세요:

**구조**:

[후킹 첫 문장]

[문제/도전 제시]

[당신의 경험/통찰]

[해결책/교훈]

[실천 가능한 조언]

[토론 유도 질문]

[해시태그]

**요구사항**:
- 1,300자 이내
- 줄바꿈 활용
- 이모지 적절히 사용
- CTA 포함
```

## 인스타그램 캡션

```
당신은 인스타그램 크리에이터입니다.

**제품/서비스/주제**: [내용]
**브랜드 톤**: [캐주얼/전문/유머]
**목표**: [참여/판매/인지]

인스타그램 캡션 3가지 버전:

### 버전 1: 스토리텔링
[감성적 스토리]

### 버전 2: 교육적
[가치 제공 + 팁]

### 버전 3: 직접적
[명확한 CTA]

**해시태그** (30개):
#해시태그1 #해시태그2 ...
```
