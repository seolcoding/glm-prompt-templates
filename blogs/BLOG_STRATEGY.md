# 멀티 플랫폼 블로그 수익화 전략

## 📊 플랫폼별 특성

| 플랫폼 | 트래픽 소스 | 광고 수익 | 자동화 | 우선순위 |
|--------|-------------|-----------|--------|----------|
| **네이버 블로그** | 네이버 검색 | 애드박스, 쇼핑윈도우 | 낮음 | ⭐⭐⭐ |
| **티스토리** | 구글 검색 | AdSense, 애드포스트 | 중간 | ⭐⭐⭐⭐⭐ |

| **Notion Blog** | 구글 검색 | 없음 (리드 생성용) | 높음 | ⭐⭐ |

## 🎯 수익 전략

### 1차: 티스토리 + AdSense (빠른 시작)
- 설정: 30분
- 승인: 1-2주
- 예상 수익: $50-200/월 (트래픽 확보 후)

### 2차: 워드프레스 + 글로벌 트래픽
- 설정: 2시간
- 비용: $5/월 (호스팅)
- 예상 수익: $100-500/월

### 3차: 네이버 블로그 (국내 트래픽)
- 설정: 1시간
- 승인: 애드박스 즉시
- 예상 수익: ₩50,000-200,000/월

## 🖼️ 이미지 전략

### 무료 이미지 소스
1. **Unsplash** (unsplash.com) - 고품질 사진
2. **Pexels** (pexels.com) - 상업적 사용 가능
3. **Pixabay** (pixabay.com) - 다양한 카테고리

### AI 이미지 생성
1. **Leonardo.ai** - 일일 150토큰 무료
2. **Microsoft Designer** - 완전 무료
3. **Canva AI** - 무료 플랜

### 이미지 워크플로
```
1. 포스팅 주제 결정
2. 키워드로 Unsplash 검색
3. 없으면 Leonardo.ai로 생성
4. Canva에서 썸네일 제작 (1200x630px)
5. 압축 (TinyPNG) 후 업로드
```

## 🔄 자동화 파이프라인

### 크로스 포스팅 워크플로
```
원본 작성 (Notion/Markdown)
    ↓
워드프레스 발행 (REST API)
    ↓
티스토리 발행 (수동/자동화)
    ↓
네이버 블로그 요약 발행
    ↓
Medium 영문 번역 발행
    ↓
소셜 미디어 홍보 (트위터, 링크드인)
```

| 화 | 네이버 | 뉴스/트렌드 |
| 수 | Medium | 영문 번역 |

## 💰 수익 계산기

### AdSense 수익 공식
```
월 수익 = (페이지뷰 / 1000) × RPM
```

### 예시 시나리오

| 트래픽 | RPM (한국) | RPM (글로벌) | 월 수익 |
|--------|-----------|--------------|---------|
| 1,000 PV | $1 | $3 | $1-3 |
| 10,000 PV | $1.5 | $5 | $15-50 |
| 50,000 PV | $2 | $8 | $100-400 |
| 100,000 PV | $2.5 | $10 | $250-1,000 |

### 목표 설정
- 3개월: 10,000 PV/월 → $50/월
- 6개월: 50,000 PV/월 → $200/월
- 12개월: 100,000 PV/월 → $500/월
