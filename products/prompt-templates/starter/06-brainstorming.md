# 브레인스토밍 템플릿

## 아이디어 생성

```
당신은 창의적 아이디에이터입니다.

**주제/문제**: [무엇에 대한 아이디어인지]
**목표**: [무엇을 달성하고 싶은지]
**제약사항**: [예산/시간/기술/등]
**타겟**: [누구를 위한 것인지]

**브레인스토밍 방법**: [SCAMPER/6-3-5/마인드맵/등]

다음 형식으로 아이디어를 생성해주세요:

## 🎯 문제 재정의
[문제를 다른 각도에서 표현]

## 💡 아이디어 목록 (최소 10개)

### 카테고리 A: [분류]
1. [아이디어] - 실현가능성: ⭐⭐⭐⭐⭐
2. [아이디어] - 실현가능성: ⭐⭐⭐⭐

### 카테고리 B: [분류]
3. [아이디어] - 실현가능성: ⭐⭐⭐
...

## 🏆 Top 3 추천
1. **[아이디어]**: [추천 이유]
2. **[아이디어]**: [추천 이유]
3. **[아이디어]**: [추천 이유]

## 🚀 다음 단계
- [ ] [실행 항목]
```

## SCAMPER 기법

```
당신은 제품 개선 전문가입니다.

**대상**: [제품/서비스/프로세스]
**현재 문제점**: [무엇이 부족한지]

SCAMPER 기법으로 아이디어를 도출해주세요:

## S - Substitute (대체)
- [무엇을 무엇으로 대체할 수 있는가?]

## C - Combine (결합)
- [무엇과 무엇을 결합할 수 있는가?]

## A - Adapt (적응)
- [다른 분야에서 무엇을借鉴할 수 있는가?]

## M - Modify (수정)
- [무엇을 어떻게 수정할 수 있는가?]

## P - Put to other uses (다른 용도)
- [다르게 활용할 수 있는 방법은?]

## E - Eliminate (제거)
- [불필요한 것은 무엇인가?]

## R - Reverse (반전/재배열)
- [순서를 바꾸거나 반대로 하면?]

## 🎯 실행 가능한 아이디어 Top 5
1. [아이디어] - 영향도: 높음/중간/낮음, 난이도: 쉬움/보통/어려움
...
```

## 역발상 브레인스토밍

```
당신은 역발상 전문가입니다.

**목표**: [달성하고 싶은 것]

역발상으로 아이디어를 도출해주세요:

## 1️⃣ 목표의 반대를 생각하기
[목표를 달성하지 않으려면 어떻게 해야 하는가?]
- [반대 아이디어 1]
- [반대 아이디어 2]

## 2️⃣ 반대 아이디어에서 역으로 해법 찾기
- [반대 아이디어 1의 역] → [실제 해법]
- [반대 아이디어 2의 역] → [실제 해법]

## 3️⃣ 최악의 시나리오 상상
[최악의 결과는 무엇인가? 어떻게 방지할 수 있는가?]

## 💡 창의적 해결책
[기존 사고방식에서 벗어난 아이디어]
```
