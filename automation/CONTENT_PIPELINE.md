# 자동화 파이프라인

## 🔄 현재 상태

| 구성 요소 | 상태 | 설명 |
|-----------|------|------|
| 콘텐츠 생성 | ⏳ API 키 대기 | ChatGPT API 준비됨 |
| 이미지 소싱 | ✅ Mock 모드 | Pexels/Unsplash API 키 필요 |
| 소셜 포스팅 | ✅ 수동 | 스크립트 준비됨 |
| 블로그 발행 | ✅ 수동 | HTML/Markdown 준비됨 |

---

## 🤖 API 키 입력 시 자동화되는 작업

### 1. 일일 콘텐츠 생성
```bash
# 매일 아침 9시 실행
- 트위터 스레드 1개 생성
- 블로그 포스트 1개 생성
- 뉴스레터 초안 1개 생성
```

### 2. 이미지 자동 검색
```
# 콘텐츠 생성 시 자동 호출
- Pexels에서 관련 이미지 검색
- 다운로드 및 최적화
- 콘텐츠에 자동 삽입
```

### 3. 소셜 미디어 예약 포스팅
```
# 최적 시간에 자동 포스팅
- 트위터: 오전 9시, 오후 7시
- LinkedIn: 화, 목, 금
- Reddit: 일, 수, 토
```

## 📅 자동화 일정

| 시간 | 작업 | 플랫폼 |
|------|------|------|
| 09:00 | 트위터 스레드 포스팅 | Twitter |
| 10:00 | 블로그 포스트 작성 | Tistory/Naver |
| 14:00 | LinkedIn 포스팅 | LinkedIn |
| 19:00 | 트위터 스레드 포스팅 | Twitter |
| 22:00 | Reddit 포스팅 | Reddit |

## 🔧 설정 필요 사항

### API 키
```env
# .env 파일에 추가
OPENAI_API_KEY=sk-xxx
PEXELS_API_KEY=your-pexels-key
UNSPLASH_ACCESS_KEY=your-unsplash-key

# 소셜 미디어 (선택)
TWITTER_API_KEY=
TWITTER_API_SECRET=
LINKEDIN_ACCESS_TOKEN=
```

### 웹훅 (선택)
```
# 블로그 발행 시 자동 알림
# 결제 발생 시 자동 알림
```
