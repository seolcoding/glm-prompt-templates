## 🧵 스레드 템플릿

### 1. 리스트형 스레드 (가장 바이럴)

```
🧵 [숫자]가지 [주제] (thread)

[후킹 첫 트윗 - 280자 이내]

[트윗 2: 첫 번째 항목]
설명 + 예시

[트윗 3: 두 번째 항목]
설명 + 예시

[트윗 4-8: 나머지 항목들]

[마지막 트윗: 요약 + CTA]
♻️ 이 스레드가 도움이 되었다면 리트윗!

[링크]
```

### 2. 스토리텔링 스레드

```
🧵 [도전/성공 스토리]

[트윗 1: 상황 설명]
[X를 하려고 했다]

[트윗 2: 문제 발생]
[하지만 Y가 문제였다]

[트윗 3: 해결책 발견]
[그래서 Z를 시도했다]

[트윗 4: 결과]
[놀라운 결과!]

[트윗 5: 교훈 + CTA]
[배운 점: ...]
```

### 3. 튜토리얼 스레드

```
🧵 [특정 기술] 단계별 가이드

[트윗 1: 문제 제기]
[많은 분들이 X를 어려워합니다]

[트윗 2-6: 단계별 설명]
각 단계를 스크린샷과 함께

[마지막 트윗: 완성 예시 + CTA]
```

## 📊 이번 주 스레드 (구체적)

### 스레드 1: "10가지 ChatGPT 프롬프트 마케터 필수"
```
🧵 마케터가 반드시 알아야 할 10가지 ChatGPT 프롬프트

시간 낭비하지 마세요. 이 프롬프트들로 마케팅 업무 10배 빠르게 👇

1/ 이메일 마케팅
"당신은 10년 차 이메일 마케터입니다.
[제품] 홍보 이메일 3가지 버전 작성:
- 스토리텔링
- 문제-해결
- 직접적 CTA"

2/ 블로그 포스트
"[주제]로 1,500자 블로그 작성.
타겟: [누구]
구조: 시작-본론-결론-CTA"

3/ 소셜 미디어
"[제품] 인스타 캡션 3가지:
- 감성적
- 교육적
- 유머러스
해시태그 20개 포함"

... (계속)
```

### 스레드 2: "AI로 1시간 만에 한 주 콘텐츠 만들기"
```
🧵 AI로 1시간 만에 한 주 치 콘텐츠 만드는 방법

워크샵 참가자들에게 공개한 실제 워크플로우입니다 👇

1/ 콘텐츠 달력 만들기 (5분)
ChatGPT: "[주제]로 한 주 블로그/소셜/이메일 주제 10개 제안"

2/ 블로그 포스트 작성 (20분)
Claude: "[주제] 1,500자 SEO 최적화 블로그"

3/ 소셜 미디어 변환 (15분)
원본: "[블로그 내용]을 인스타/링크드인용으로 변환"

4/ 이메일 뉴스레터 (10분)
"[이번 주 콘텐츠] 뉴스레터 형식으로 요약"

5/ 이미지 생성 (10분)
Midjourney/DALL-E로 썸네일 생성

총 60분으로 한 주 치 콘텐츠 완성 ✅
```

## 🔗 크로스 포스팅 전략

### 스레드 → 다른 플랫폼
1. **트위터 스레드** (원본)
2. → **LinkedIn 포스트** (확장 버전)
3. → **티스토리 블로그** (상세 버전)
4. → **Medium** (영문 번역)
5. → **인스타 캐러셀** (시각화)

### 순서
```
트위터 발행
  ↓ 24시간 후
LinkedIn 발행
  ↓ 48시간 후
블로그 발행
  ↓ 24시간 후
Medium 발행
```
