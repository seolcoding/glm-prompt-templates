# KPI 프레임워크

## 핵심 성과 지표

### 1. 발견 단계 (Discovery)
| 지표 | 정의 | 목표 |
|------|------|------|
| 아이디어 수 | 총 아이디어 개수 | ≥10개/월 |
| 점수 매겨진 아이디어 | 5개 영역에서 평가된 아이디어 | 100% |
| 연구된 소스 | 공개 연구 소스 수 | ≥5개/아이디어 |

### 2. 전환 단계 (Conversion)
| 지표 | 정의 | 목표 |
|------|------|------|
| 방문자 수 (Visitors) | 총 방문 수 | 측정 중 |
| 옵트인율 | 리드 캡처 / 방문자 | ≥2% |
| 구매 시작율 | 체크아웃 시작 / 옵트인 | ≥10% |
| 완료율 | 구매 완료 / 시작 | ≥50% |

### 3. 수익 단계 (Revenue)
| 지표 | 정의 | 목표 |
|------|------|------|
| 평균 주문 가치 (AOV) | 총 수익 / 주문 수 | 측정 중 |
| 고객 획득 비용 (CAC) | 마케팅 비용 / 신규 고객 | <AOV |
| 월간 반복 수익 (MRR) | 구독 수익 | 측정 중 |

### 4. 운영 단계 (Operations)
| 지표 | 정의 | 목표 |
|------|------|------|
| 빌드 비용 | 개발 시간 × 시간당 비용 | 최소화 |
| 런타임 비용 | 월간 서버/API 비용 | <수익의 10% |
| 실험 속도 | 실행된 실험 수 / 월 | ≥3개 |

## 데이터 수집 방법
- 클라이언트 사이드: JavaScript 이벤트 추적
- 서버 사이드: API 로깅
- 분석: /apps/admin 대시보드
