## 10가지 체크리스트

### ✅ 역할 부여
- [ ] "당신은 [전문가 역할]입니다"로 시작
- [ ] 구체적인 역할 명시 (예: "10년 경력의 마케터")

### ✅ 컨텍스트 제공
- [ ] 배경 정보 포함
- [ ] 타겟 독자/청중 명시
- [ ] 목표 명확히

### ✅ 출력 형식 지정
- [ ] 원하는 구조 설명
- [ ] 길이 제한 설정
- [ ] 포맷 지정 (텍스트/표/JSON/코드)

### ✅ 예시 포함
- [ ] Few-shot 예시 1-3개 제공
- [ ] 예상 출력 샘플 보여주기

- [ ] 스타일 가이드 제공

### ✅ 제약사항 명시
- [ ] 하지 말아야 할 것 명시
- [ ] 피해야 할 내용 지정
- [ ] 톤/스타일 제한

### ✅ 단계별 요청
- [ ] 복잡한 작업을 단계로 분해
- [ ] 각 단계별 확인 요청
- [ ] 점진적 개선 요청

### ✅ 검증 요청
- [ ] 결과물 검토 요청
- [ ] 사실 확인 요청
- [ ] 논리 검증 요청
- [ ] 출처 확인 (필요시)

### ✅ 반복 개선
- [ ] "더 [간결/상세/전문적]하게" 요청
- [ ] 특정 부분 수정 지시
- [ ] 대안 제안 요청

### ✅ 톤 지정
- [ ] 전문적/친근함/유머러스러스
- [ ] 한국어 존댓말/반말 선택
- [ ] 일관된 톤 유지

### ✅ 길이 확인
- [ ] 적절한 길이인지 확인
- [ ] 필요시 확장/축소 요청
- [ ] 가독성을 위한 문단 구분
