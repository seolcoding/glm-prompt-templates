# 마케터를 위한 10가지 필수 ChatGPT 프롬프트

> **업데이트**: 2026년 최신 버전 기준으로 테스트된 프롬프트입니다.

## 🎯 왜 마케터에게 ChatGPT가 필요한가?

마케팅은 끊임없는 콘텐츠 생산이 필요합니다. 블로그 포스트, 이메일, 소셜 미디어, 광고 카피... 매일 수십 개의 콘텐츠가 필요하죠.

ChatGPT와 Claude 같은 AI 도구는 마케터에게 **시간을 절약**해주고 **일관된 품질**을 보장합니다. 하지만, "AI한테 써봐"라고만 하면 평범한 결과만 나옵니다.

핵심은 **정확한 프롬프트**입니다.

## 📝 10가지 필수 ChatGPT 프롬프트

### 1. 블로그 포스트 작성

```
당신은 10년 경력의 콘텐츠 마케터입니다.

[주제]에 대한 1,500자 블로그 포스트를 작성해주세요.

**타겟 독자**: [구체적 타겟]
**톤**: [전문적/친근함]
**구조**:
1. 흥미로운 시작 (통계나 질문)
2. 3-5개 섹션
3. 실천 가능한 결론
4. CTA

**요구사항**:
- SEO 키워드 자연스럽게 포함
- 소제목 사용
- 읽기 쉬운 문단
```

### 2. 이메일 마케팅

```
[제품/서비스]를 홍보하는 이메일 캠페인을 작성해주세요.

**이메일 시퀀스**: 3개
1. 인지 (Awareness) - 문제 제기
2. 고려 (Consideration) - 해결책 제시
3. 행동 (Action) - 할인 + 긴급성

각 이메일:
- 제목: 클릭 유도형
- 본문: 200자 이내
- CTA: 명확하게
```

### 3. 인스타그램 캡션

```
[제품]에 대한 인스타그램 캡션 3가지 버전:

1. **스토리텔링** - 감정적 연결
2. **교육적** - 가치 제공
3. **직접적** - 명확한 CTA

**요구사항**:
- 125자 이내
- 관련 해시태그 20개
- 이모지 적절히 사용
```

### 4. 광고 카피 (A/B 테스트용)

```
[제품]에 대한 Facebook/Instagram 광고 카피를 작성해주세요.

**A/B 테스트용 2개 버전**:

버전 A: 문제 해결 중심
버전 B: 이익 중심

각 버전:
- 헤드라인: 25자 이내
- 본문: 90자 이내
- CTA 버튼 텍스트
```

### 5. 제품 설명 페이지

```
[제품]의 랜딩 페이지 카피를 작성해주세요.

**구조**:
1. 히어로 섹션 - 1문장 가치 제안
2. 문제 섹션 - 타겟의 pain points 3개
3. 솔루션 섹션 - 제품이 어떻게 해결하는지
4. 특징 섹션 - 핵심 기능 3-5개
5. 사회적 증거 - 후기/수치
6. CTA 섹션

**톤**: [전문적/친근함]
```

### 6. 경쟁사 분석

```
[산업/제품 카테고리]의 경쟁사 분석을 수행해주세요.

**분석 항목**:
- 주요 경쟁사 5개
- 각사의 강점/약점
- 가격 전략
- 마케팅 채널
- 차별화 기회

**출력 형식**: 표 형태
```

### 7. 고객 페르소나

```
[제품/서비스]의 타겟 고객 페르소나 3개를 작성해주세요.

각 페르소나:
- 이름/나이/직업
- 목표 2개
- Pain points 3개
- 정보 소스 (어디서 정보를 얻는지)
- 구매 결정 요인
- 반대 요인 (구매하지 않는 이유)
```

### 8. 콘텐츠 캘린더

```
[월] [제품/브랜드]의 소셜 미디어 콘텐츠 캘린더를 작성해주세요.

**플랫폼**: [인스타/링크드인/트위터]
**주제**: [주제]

**출력**:
| 날짜 | 플랫폼 | 콘텐츠 유형 | 주제 | 캡션 아이디어 |
|-----|-------|------------|------|--------------|
```

### 9. SEO 메타 데이터

```
[페이지 주제]의 SEO 메타 데이터를 작성해주세요.

**타겟 키워드**: [키워드]

**출력**:
- **Title**: 60자 이내, 키워드 포함
- **Meta Description**: 160자 이내, 클릭 유도
- **H1**: 페이지 제목
- **URL 슬러그**: 짧고 키워드 포함
- **이미지 Alt 텍스트**: 3개
```

### 10. AIDA 프레임워크 광고

```
[제품]에 대한 AIDA 프레임워크 광고를 작성해주세요.

**AIDA**:
1. **Attention** - 주의 끌기
2. **Interest** - 흥미 유발
3. **Desire** - 욕구 생성
4. **Action** - 행동 유도

**플랫폼**: [Facebook/LinkedIn/인스타]
**길이**: 플랫폼에 맞게
```

## 💡 프롬프트 팁

1. **구체적으로**: "좋은 글" 대신 "1,500자 블로그 포스트"
2. **맥락 제공**: 타겟 독자, 톤, 목표 명시
3. **반복 개선**: 첫 결과를 보고 "더 [간결하게/상세하게/전문적으로]" 요청
4. **예시 추가**: "다음과 같은 스타일로: [예시]"
