# Gumroad 결제 설정 가이드

## 1. Gumroad 계정 생성
1. https://gumroad.com 접속
2. "Start selling" 클릭
3. 이메일로 가입

## 2. 제품 등록

### 무료 (Free)
- **제품명**: 프롬프트 작성 10계명 체크리스트
- **가격**: 무료
- **파일**: `products/prompt-templates/free/prompt-checklist.md`
- **설명**: AI 프롬프트 작성의 핵심 원칙을 담은 1페이지 체크리스트

### 스타터 팩 ($9)
- **제품명**: 50개 필수 AI 프롬프트 템플릿 - 스타터
- **가격**: $9
- **파일**: `products/prompt-templates/starter/` (10개 템플릿)
- **설명**: 가장 많이 사용하는 10개 프롬프트 템플릿

### 코어 팩 ($29)
- **제품명**: 50개 필수 AI 프롬프트 템플릿 - 코어
- **가격**: $29
- **파일**: 전체 50개 템플릿
- **설명**: 모든 50개 프롬프트 템플릿 + 업데이트

### 프리미엄 세션 ($99)
- **제품명**: 프롬프트 커스터마이징 1:1 세션
- **가격**: $99
- **형식**: 30분 화상 통화
- **설명**: 당신의 업무에 맞는 커스텀 프롬프트 제작

## 3. 결제 링크 형식
```
https://gum.co/[PRODUCT_ID]
```

## 4. 랜딩 페이지에 연결
Gumroad에서 제공하는 "Overlay" 또는 "Redirect" 방식 사용
