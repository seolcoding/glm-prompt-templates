# 비전 (Vision)

## 미션
법적이고 윤리적인 자동화 수익 실험 시스템을 구축하여 디지털 수익 기회를 발견, 실행, 측정, 개선합니다.

## 핵심 가치
- **준수 우선**: 모든 활동은 법적, 윤리적, 플랫폼 규칙을 준수
- **학습 속도**: 빠른 실험을 통한 검증된 학습
- **저비용 운영**: 최소한의 운영 비용으로 시작
- **데이터 기반**: 모든 결정은 측정 가능한 데이터에 기반

## 허용 비즈니스 모델
1. 디지털 제품 (Templates, E-books, Courses)
2. 뉴스레터 및 구독 서비스
3. 마이크로 SaaS
4. 제품화된 서비스
5. B2B 리드 생성 (명확한 옵트인 포함)
6. 컴플라이언스 제휴 (적절한 공개 포함)

## 금지 사항
- 사기, 스팸, 가짜 리뷰
- 저작권 침해
- 플랫폼 ToS 위반
- 개인정보 무단 수집
- 기만적 마케팅

## 성공 지표
- 월간 활성 사용자 수
- 전환율
- 고객 획득 비용 (CAC)
- 생애 가치 (LTV)
- 순수익
