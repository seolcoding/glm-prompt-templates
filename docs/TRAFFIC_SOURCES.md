## 🎯 즉시 실행 가능한 작업

### 1. 트위터 포스팅 (수동)
```bash
# 터미널에서 실행
cat content/twitter/ready-to-post-001.md | pbcopy
open https://twitter.com/compose/tweet
# ⌘+V로 붙여넣고 포스팅
```

### 2. Ko-fi 설정
1. https://ko-fi.com 가입
2. Shop → Add Digital Product
3. 이름: AI 프롬프트 스타터 팩
4. 가격: $9
5. 파일: products/starter-pack.zip
6. 링크를 apps/web/index.html의 YOUR_KOFI에 입력
7. `vercel --prod`로 재배포

### 3. LinkedIn 포스팅
```bash
cat content/linkedin/post-001.md | pbcopy
open https://www.linkedin.com/feed/
```

### 4. Reddit 포스팅
```bash
open https://www.reddit.com/r/ChatGPT/submit
```
